The Standard Library

The StackLang standard library is composed of two parts - the extended command set for primitive types (see individual primitive type pages for more information), and this set of utility commands. By default both parts are automatically included, but running the interpreter with the -b option will turn off this automatic inclusion. Both parts are written in StackLang, and can be found in libs/std.sta, as well as files included from that file. (The official implementation splits off many of the extended command set into separate files.)

Utility Functions

These functions are provided as being commonly used. They are not associated with any particular type, but are generally useful.

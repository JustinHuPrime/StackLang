Special Primitive Functions

These functions have special side effects outside of the usual evaluation model.

define : Substack(Type) Substack(Any) Command(Quoted) ->
Defines a function in the global context.

define* : Substack(Type) Substack(Any) Command(Quoted) Command(Quoted) ->
Defines the second command in the context of the first command (a "local" function). Names, however, are always global, so the name must still be unique.

undefine : Command(Quoted) ->
Undefines the command. May not undefine primitive commands, but anything defined using define or define* can be undefined.

include : String ->
Looks for:
  1. the named file within a 'libs' folder within the directory that the stacklang executable is in,
  2. the named file within 'libs', but adding a .stl extension,
  3. the named file from the current directory,
  4. the named file from the current directory, but adding a .stl extension
then reads the file line by line, feeding it through the preprocessor first, and pushes each element onto the stack, evaluating as it goes.

drop : Any ->
Removes one element from the stack.

rotate : Number(Exact) ->
Takes the n'th element of the stack and moves it to the active end.

rotate* : Number(Exact) Number(Exact) ->
Takes the (second number)'th element of the stack and moves it to the active end (first number) times.

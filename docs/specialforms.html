Special Primitive Functions

These functions have special side effects outside of the usual evaluation model.

define : Command Substack Substack(Type) ->
Defines a function in the global context.

define* : Command Command Substack Substack(Type) ->
Defines the second command in the context of the first command (a "local" function). Names, however, are always global, so the name must still be unique.

undefine : Command ->
Undefines the command. May not undefine primitive commands, but anything defined using define or define* can be undefined.

include : String ->
Looks for:

  1. the named file from the current directory or absolute path,
  2. the named file from the current directory or absolute path, but adding a .stl extension
  3. the named file within a 'libs' folder from the current directory,
  4. the named file within 'libs', but adding a .stl extension,
then reads the file line by line, feeding it through the preprocessor first, and pushes each element onto the stack, evaluating as it goes. If the file does not exist, it fails with a RuntimeError.

drop : Any ->
Removes one element from the stack.

drop* : Number ->
Takes n elements and removes them from the stack.

clear : ->
Clears the stack, removing all elements from the stack.

rotate : Number ->
Takes the n'th element (1 based, starting after the parameter) of the stack and moves it to the active end.

rotate* : Number Number ->
Takes the (second number)'th element of the stack and moves it to the active end (first number) times.

duplicate : Any -> Any Any
Duplciates the given element.

error : String ->
Raises a RuntimeError with the specified message.

error* : Number String String ->
Raises a RuntimeError with the specified context (first string and number) and message (second string). Will cause a RuntimeError itself if the number is not valid.

null : ->
This command does nothing, but does cause the evaluator to evaluate any unquoted commands after it. Useful for recovering from an error.

identity : Any -> Any
This command produces its input.
